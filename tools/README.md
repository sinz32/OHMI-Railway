# 오미 철도 운행 정보 - 도구

## crawler.py
- [오미 철도 홈페이지](https://www.ohmitetudo.co.jp/railway/ride/timetable/)로부터 시간표 정보를 크롤링하여, `timetable.json` 파일로 저장
- 주석 처리된 `url` 부분을 조절하여 다른 노선의 정보 크롤링 가능
- 종점 도착 시간은 정보가 없기 때문에 정보가 수집되지 않음

## csv2json.py
- `crawler.py`로 생성한 `timetable.json` 파일을 `.csv` 파일들로 변환
- 변환 전 `timetable.json` 파일에서 빈 배열을 선언하는 부분을 삭제해야 작동함 (`up: []`, `down: []`과 같은 부분)
- 변환 결과물에 오류가 있으니 수정 필요
  - `상행 열차들의 처음 역 출발 시간`이 `하행 시간표`의 가장 아래에 저장됨
  - `하행 열차들의 처음 역 출발 시간`이 `상행 시간표`의 가장 아래에 저장됨
- 오미 철도의 열차 운행 다이어그램은 복잡하여, 크롤링만으로는 완성할 수 없기 때문에, .pdf로 되어있는 전선 시간표를 보면서 하나하나 수정해야 함

## Q. 왜 버그를 수정하지 않았나요?
- A. 한 번만 실행하고 끝낼 소스이며, 디버깅에 걸리는 시간보다 사람이 직접 결과물을 수정하는 것이 더 빠르기 때문입니다.
